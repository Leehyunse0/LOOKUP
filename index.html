LOOKUP는 이미지에서 스타일 어휘를 추출해, 누구나 검색 가능한 언어로 제공합니다.

Problem

패션 언어는
아는 사람만 안다.

소비자는 이미지를 보고 느낌을 알지만,
검색창 앞에 서면 단어가 떠오르지 않습니다.
레퍼런스는 넘치지만,
내 취향에 딱 맞는 단어를 찾기는 더 어려워졌습니다.

01

이미지 발견

인스타그램 · 핀터레스트 · 쇼핑몰에서 마음에 드는 룩을 저장합니다.

02

검색 시도

“하얀 치마 발레 느낌”, “레이스 스커트 단정” 같은 모호한 키워드를 입력합니다.

03

검색 실패

원하던 실루엣과 디테일이 아닌 수많은 상품 결과에 지칩니다.

04

지인에게 의존

“이런 거 뭐라고 검색해?” 패션에 밝은 친구나 커뮤니티에 물어봅니다.

05

결국 우연히 도달

몇 번의 시행착오 끝에야 ‘발레코어’, ‘메리제인’ 같은 정답 단어를 알게 됩니다.

Solution

이미지 한 장으로 완성하는
패션 언어 인덱스.

스타일 태깅

룩의 무드와 실루엣을 중심으로, 사람이 쓰는 언어에 가깝게 태그를 구성합니다.

Ballet core Office casual Romantic minimal

아이템 명칭

디테일과 소재를 반영해 상품명에 쓸 수 있는 명칭을 제안합니다.

레이스 트리밍 미디 스커트 스트랩 메리제인 플랫 시어 리본 블라우스

검색 키워드 제공

플랫폼·국가에 따라 다른 검색 패턴을 고려해 키워드를 추천합니다.

"발레코어 스커트" "lace trim skirt" "메리 제인 플랫슈즈"

Target

LOOKUP이 먼저 돕고 싶은
사람들.

입문자 (Primary)

패션은 좋아하지만, 전문 용어나 브랜드 맥락은 아직 낯선 사람들.
이미지 중심으로 레퍼런스를 모으지만, 검색·구매로 잘 이어지지 않습니다.

  • 레퍼런스를 많이 저장하지만, 나중에 다시 찾기 어렵다.
  • 쇼핑할 때 '뭐라고 검색해야 할지'가 가장 막막하다.
  • 패션 언어를 자연스럽게 익히고 싶다.

고관여자 (Secondary)

이미 패션 언어에 익숙한 에디터, MD, 브랜드, 스타일리스트.
자신의 어휘를 더 많은 사람과 나누고, 검색 구조를 개선하고 싶어 합니다.

  • 자신의 큐레이션을 더 넓은 대중과 연결하고 싶다.
  • 브랜드 세계관을 해치지 않는 검색 구조가 필요하다.
  • 내가 아는 단어를, 아직 모르는 사람들에게 다리 놓고 싶다.

패션 언어를 빠르고 쉽게 — LOOKUP은 두 집단 사이의 사전을 만듭니다.

Roadmap

LOOKUP의 단계별 여정.

Phase 1 · v1.0

패션 언어 획득

이미지 기반 스타일 태깅, 아이템 명칭, 핵심 검색 키워드를 정교하게 수집·정리합니다.

  • 코어 스타일·실루엣 태그 라이브러리 구축
  • 국·영문 패션 어휘 매핑
  • 입문자 기준의 설명형 태그 실험

Phase 2 · v1.5

레퍼런스 확장

런웨이, 에디토리얼, 인플루언서 룩북 등 레퍼런스를 확장해 문맥을 더 풍부하게 만듭니다.

  • 브랜드·쇼핑몰·커뮤니티 이미지 연동
  • 코디 단위 레퍼런스 추천
  • 사용자별 스타일 맵 시각화

Phase 3 · v2.0

패션의 민주화

패션 언어를 소수의 전문 영역에서 꺼내, 누구나 접근 가능한 공용 인프라로 만듭니다.

  • 플랫폼·브랜드를 아우르는 검색 언어 계층 제공
  • 개인 스타일 어휘장 생성 및 공유
  • 교육·커뮤니티와 연계한 오픈 데이터셋
